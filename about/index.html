🌟 博客简介

🚀 本地开发

环境要求

  • Node.js 18+
  • Git

安装步骤

# 克隆仓库
git clone https://github.com/Alger0001/hexo-blog.git 
cd hexo-blog

# 安装依赖
npm install

# 启动本地服务器
npm run server

# 访问 http://localhost:4000
